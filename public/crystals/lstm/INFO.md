# THE MEMORY CELL

**Architecture:** LSTM (Long Short-Term Memory)
**Shape:** The Helix Loop
**Concept:** Persistence

The first neural network that could "remember".
Invented in **1997**, it solved the vanishing gradient problem for time-series data.

**Use Cases:**
*   **Translation:** Google Translate used LSTMs for years.
*   **Speech Recognition:** Powering Siri and Alexa before Transformers took over.
